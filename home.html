gcloud

gcloud-node is the Google Cloud Client Library for Node.js - an idiomatic, intuitive, and natural way for Node.js developers to integrate with Google Cloud Platform services, like Cloud Datastore and Cloud Storage.

One-line npm install

$ npm install --save gcloud

What is it?

The philosophy behind gcloud-node is that it reduces boilerplate and time-to-start. High-level API abstractions and a focus on the most common use-cases makes it lightweight yet still effective. It is also preconfigured to access Google Cloud services and authorization is handled for you, whether your code runs on the Google Cloud Platform.

Moreover, gcloud-node embraces the idioms of Node.js, like returning a Readable stream or returning an error as the first object in a callback.

Sample code

var gcloud = require('gcloud'); var datastore = gcloud.datastore; // new datastore.Dataset(); var storage = gcloud.storage; // new storage.Bucket();
